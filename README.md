# setup-computer

Bootstrap a new macOS machine with Homebrew, Bash, and common dev tools.

## Requirements

- macOS 12 (Monterey) or later

## Quick Start

```bash
git clone <this-repo> && cd setup-computer
chmod +x setup_new_mac.sh
./setup_new_mac.sh --all
```

See [NEW_MAC_SETUP.md](NEW_MAC_SETUP.md) for full details and options.
